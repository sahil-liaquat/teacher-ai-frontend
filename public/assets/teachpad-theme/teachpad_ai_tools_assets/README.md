# TeachPad AI Tools UI Assets

This ZIP contains the colours and icon assets used in the TeachPad AI Tools page design.

## Folders

- `icons/` — raw line SVG icons.
- `pastel-icon-tiles/` — ready-to-use pastel rounded-square SVG icon tiles matching the UI style.
- `colors.json` — colour palette as JSON tokens.
- `teachpad-colors.css` — CSS variables for the palette.
- `icon-manifest.json` — mapping of icon names, colours, backgrounds and file paths.

## Note

The React page uses `lucide-react` icon component names. The SVGs included here are matching pastel-style export assets for design handoff and quick usage.
